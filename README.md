# Parth Maheshwari | Portfolio Website

Welcome to the repository for my professional portfolio website! This site is designed to showcase my experience, projects, skills, and certifications as a **Data Analyst / Analytics Engineer** and **Data Scientist**.

🔗 **Live Link:** [https://parthm16.github.io/](https://parthm16.github.io/)

---

## 🚀 Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Custom Vanilla CSS3 (Dark-mode, Glassmorphism elements, Fluid grids)
- **Interactions:** Vanilla JavaScript (ES6+)
- **Hosting:** GitHub Pages

---

## ✨ Features

- **Responsive Grid System:** Adapts perfectly across mobile, tablet, and desktop screens.
- **Dynamic Experience Tabs:** Toggle between roles to see tailored experience timelines and relevant technical skill lists.
- **Interactive Skills Section:** Visual highlight effects and micro-animations styled dynamically to match the role color scheme.
- **Live Impact Ticker:** Animated count-up counter that displays key performance metrics (e.g., query speedups, record migrations, revenue impact) when scrolled into view.
- **Optimized Performance:** Fast load times with minimal assets, zero third-party framework dependencies, and smooth scroll transitions.

## 🛠️ Local Development

To run the site locally, clone this repository and open the `index.html` file in any modern browser:

```bash
# Clone the repository
git clone https://github.com/ParthM16/ParthM16.github.io.git

# Navigate into the folder
cd ParthM16.github.io

# Open index.html (or use a local development server like VS Code Live Server)
start index.html
```
